我用了dp的方法，有一个坑的地方就是可能整个数组全是负数。

Solution2:
在线处理，如果i号元素加起来比
```c++
class Solution {
public:
    int maxSubArray(vector<int>& nums) {
        int res = INT_MIN, curSum = 0;
        for (int num : nums) {
            curSum = max(curSum + num, num);
            res = max(res, curSum);
        }
    }
};
```
